# This is basic example of an Express app

## About the app

**Routes**
    
        / - Home page
        /:pokemon
            - Get pokemon data
            - example: /pikachu - returns pikachu data


## File structure

    .
    ├── README.md
    ├── index.js
    ├── package.json
    ├── package-lock.json


## Installation
    
        `npm install`


## Run the app

        `npm run dev`

## Run the app in production mode
    
            `npm start`
